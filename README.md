# CodeAlpha Basic Network Sniffer

## Description
This project is a simple Python-based network sniffer that captures live network packets and displays:
- Source IP
- Destination IP
- Protocol (TCP/UDP/ICMP)
- Port numbers
- Packet payload (if available)

## Features
- Real-time packet capture
- Protocol detection
- Lightweight output
- Easy to run

## Installation
pip install -r requirements.txt

## Run
python network_sniffer.py

## Warning
Run with administrator/root privileges for packet capture.
